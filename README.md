## Cara Menjalankan Program (vscode)
- Buka src/app
- Run MAIN.c
- Apabila splash screen terlihat tidak jelas maka coba jalankan di mode debug dalam vscode

## Cara Menjalankan Program (terminal)
- cd src
- cd App
- gcc MAIN.c -o getEXE
- .\getExe
